# Screener

For a specific stock list, creates a CSV with the following fields:

1. Ticker
2. Sector
3. Market Cap, current
4. EV, current with TTM values for Debt and Cash
5. Debt Metrics
   1. Current Ratio
   2. Debt To Equity
6. Performance Metrics
   1. RoA
   2. RoE
   3. RoIC
7. Valuation
   1. P/E
   2. EV/EBIT
   3. EV/FCFF
   4. Intrinsic Value
